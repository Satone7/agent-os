# Loop Break Prompt

You are receiving this message because the main agent detected that you may be stuck in a loop.

## What This Means

You have been performing similar actions repeatedly without making meaningful progress. This can happen when:
- The same error keeps occurring
- You're trying the same approach multiple times
- You're unclear about the requirements

## How to Break the Loop

1. **Stop and reassess**: Take a step back and think about the overall goal
2. **Try a different approach**: If one method isn't working, try another
3. **Ask for clarification**: If requirements are unclear, ask for more information
4. **Break down the task**: Split complex tasks into smaller, manageable pieces
5. **Focus on progress over perfection**: It's better to make progress than to be stuck

## Specific Suggestions

Based on the detected pattern, here are specific suggestions:

- Review the original requirements carefully
- Consider alternative tools or methods
- Check if there are any blockers you haven't addressed
- Provide a summary of what you've tried and what didn't work

## Remember

The main agent is here to help. If you're truly stuck, communicate the issue clearly and suggest what help you need.
